# CU_Meet_project
A simple meeting room (not real) booking application for CU students built with SwiftUI.

To run this code, you'll need to have a Firebase Firestore database and `GoogleService-Info.plist` with all the credentials (unstaged in this repo to prevent API leaks).

## Setup
- [Firebase Console](https://console.firebase.google.com/)
